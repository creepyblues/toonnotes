## Expansion Prompts Library

### "Go Deeper" Prompts
- "Why does this matter to you personally?"
- "When did you first notice this problem?"
- "What would the world look like if this existed?"

### "Flip It" Prompts
- "What if the opposite approach worked better?"
- "What would happen if you made it 10x simpler?"
- "Who would hate this idea and why?"

### "Audience" Prompts
- "Who feels this pain most acutely?"
- "When do they experience this frustration?"
- "What do they currently do instead?"

### "Mechanics" Prompts
- "What's the simplest version that could work?"
- "What's the first thing a user would do?"
- "What data would you need?"

### "Risks" Prompts
- "What could make this completely fail?"
- "Why hasn't someone built this already?"
- "What's the hardest part to get right?"
